I'm the File From Folder02

Look at the address bar. It says I'm in the directory FakeRoute. But I'm not. I live in Folder02: 

../Folder02/Other.html

Our call to app.use reroutes the call to FakeRoute to the real directory, but no one on the client side would have a way to know it is happening.

Now load index.html, but don't include Public in its path. We would think this call should fail, since index.html is in this directory:

../public/index.html

But the call succeeds because of the way we employed app.use.
