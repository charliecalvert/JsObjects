# Angular Three Modules

To run the program:

<pre>
bower install
python3 http.server 30025
http://localhost:30025
</pre>

To run the tests:

<pre>
bower install
python3 http.server 30025
http://localhost:30025/TestBoats.html
</pre>

Get the [latest Jasmine CDN](https://cdnjs.com/libraries/jasmine).
