# Angular Routing View

## Usage

To run

<pre>
npm install
bower install
node server.js
</pre>

If they are unsure which version of angular to pick, take the latest.

```

    3) angular#^1.5.3 which resolved to 1.5.3 and is required by angular-routing-view
```
