This layout is based very closely on the twelve column example from MDN. Note that the canonical example looks and acts the same but is simpler.

We try to avoid reinventing the wheel. As a rule, most of the layouts you need have been created by someone else. Find where they are done right, and use them.

In this layout, we display the areas in source order for any screen less that 500 pixels wide. We go to a two column layout, and then to a three column layout by redefining the grid, and the placement of items on the grid.

Sidebar: Elvenware and JsObjects are the places where Charlie Calvert keeps notes and examples.
